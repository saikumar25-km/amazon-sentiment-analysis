# Amazon Product Sentiment Analysis

# Project Overview

This project analyzes customer reviews of three products:

* Apple iPad
* Samsung Galaxy Tab
* OnePlus Pad

##  Objective

To classify customer reviews into Positive, Negative, and Neutral sentiments.

## 🔧 Steps Performed

### 1. Data Collection

* Used dataset with 15,000 reviews

### 2. Data Cleaning

* Removed duplicates
* Handled missing values

### 3. Text Preprocessing

* Converted text to lowercase
* Removed punctuation
* Removed stopwords

### 4. Sentiment Analysis

* Used TextBlob to calculate sentiment score
* Classified reviews into Positive, Negative, Neutral

### 5. Data Analysis

* Compared sentiment across products
* Calculated average ratings

### 6. Dashboard (Power BI)

Created:

* Pie chart (Sentiment Distribution)
* Bar chart (Product vs Sentiment)
* Cards (Total, Positive, Negative Reviews)
* Average Rating Chart
